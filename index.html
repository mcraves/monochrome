Import from CSV

Please use Exportify to export your Spotify playlist into a .csv.

Please use TuneMyMusic to export your Apple Music playlist into a .csv.

Paste a YouTube Music Playlist URL.

Make sure its headers are in English.

Strict Album Matching Album name should strictly match CSV metadata.

Import from JSPF

JSPF (JSON Shareable Playlist Format) is supported by many media players.


Import from XSPF

XSPF (XML Shareable Playlist Format) is a standard XML playlist format.


Import from XML

Import playlists from generic XML formats including iTunes XML playlists.


Import from M3U/M3U8

M3U is the most widely supported playlist format.


Warning: This feature isn't perfect and is prone to errors! Please check your playlist after import.
